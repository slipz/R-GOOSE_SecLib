Function that decodes 2 bytes from a specific index to an integer.

This function receives the target buffer (buffer) and position (index), and decodes the value the following 2 bytes, returning its value as an integer.

Below is and example of usage:

int timeToNextKey;
timeToNextKey = decode_2bytesToInt(buffer, INDEX_TIMENEXTKEY);
Parameters
bufferPointer (uint8_t*) containg the target buffer
indexVariable (int) containg the starting index of buffer from where the integer should be decoded
Returns
The function returns an integer containing the value decoded
Warning
This functions doesn't perform any kind of input validation, more precisely on index.
